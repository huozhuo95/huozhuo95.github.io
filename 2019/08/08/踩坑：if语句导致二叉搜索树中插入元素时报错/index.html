踩坑：if语句导致二叉搜索树中插入元素时报错

前言

​ 二叉搜索树是一种特殊的二叉树，特点是根节点的左子节点永远比它小，而它的右子节点永远比它大。用二叉搜索树进行查找和插入速度都很快，因此是一种常用的数据结构。

​ 由于它的特性，在实现二叉搜索树的数据结构的过程中，比如插入、搜索的功能，会在成员函数的实现里用到很多if语句，用来判断元素的大小。本文针对if语句造成的问题进行分析，以此避免今后类似的错误。

问题
