C++迭代器的总结

C++迭代器是用来遍历容器（vector，set，list，map等）的工具，也可以用来遍历string，可以把它当作指针来理解，根据不同的容器，用法如下：

vector

获取迭代器

对于一个已经初始化了的vector对象，可以通过以下语句获取迭代器：

其中：

for (auto iter = vt.begin(); iter!=vt.end(); iter++)
{
cout << *iter << endl;
}

for ( vector:: reverse_iterator iter = vt.rbegin();
iter!=vt.rend(); iter++)
{
cout << *iter << endl;
}


其中：

```vector<int>:: reverse_iterator```为逆向遍历的迭代器的类型；

```vt.rbegin()```为最后一个元素；

```vt.rend()```为第一个元素的下一个位置（可以理解为前一个位置）

**注意！！**

```iter```仍然每次增加一，不要认为是逆序遍历就每次减一
